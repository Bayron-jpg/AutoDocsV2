# AutoDocs
* Automatizador de plantillas para documentos académicos.

## Tecnologías
| Dependencia   | Uso               |
| ----------    |----------         |
| Python-docx   | Documentos        |
| CustomTKinter | Interfaz          |
| Docx2Pdf      | Transformar a PDF |
| PyInstaller   | Comprimir a .exe  |
